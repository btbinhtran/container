## Running Tests

```
git clone git://github.com/viatropos/tower-container.git
cd tower-container
npm install
```

then run the tests:

```
npm test
```

Requires `mocha`, `chai`, and `phantomjs`:

```
brew install phantomjs
npm install mocha -g
npm install phantomjs chai
```
